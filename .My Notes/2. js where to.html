Old JavaScript examples may use a type attribute: <script type="text/javascript">. The type attribute is not required. JavaScript is the default scripting language in HTML.

JavaScript Functions and Events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for. For example, a function can be called when an event occurs, like when the user clicks a button.

Note: JavaScript functions are executed in the order they are called. Note: Placing scripts at the bottom of the <body> element improves the display speed, because script interpretation slows down the display.

JavaScript in External file

External scripts are practical when the same code is used in many different web pages. You can place an external script reference in <head> or <body> as you like as below: <script src="myScript.js"></script>
External scripts cannot contain <script> tags.

External JS References

(myFunction is stored in "myScript.js")

An external script can be referenced in 3 different ways:

  • With a full URL (a full web address)

    • This example uses a full web URL to link to "myScript.js".
    <script src="https://www.w3schools.com/js/myScript.js"></script>

  • With a file path (like /js/)

    • This example uses a file path to link to "myScript.js".
    <script src="/js/myScript.js"></script>

  • Without any path

    • This example links to "myScript.js".
    <script src="myScript.js"></script>
